# nlp-m3-NER-news
NER project on news articles (CoNLL-2003) for Module 3 NLP - pipeline, evaluation and learning

## Structure
- `data/raw/` : original CoNLL-2003 dataset files
- `data/processed/` : preprocessed/tokenized text
- `notebooks/` : Jupyter notebooks for experimentation and main NER workflow
- `src/` : Python scripts for preprocessing, training, and evaluation
- `requirements.txt` : Python dependencies
- `.gitignore` : ignored files (cache, data/raw, etc.)

## Usage
1. Open the repository in GitHub Codespace or VS Code.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the notebook `notebooks/NER_news.ipynb` for the full NER pipeline.
4. Explore scripts in `src/` for modular functionality.
